```

│   tapi seberapa banyak yang kamu pelajari               │

│                                                         │
└─────────────────────────────────────────────────────────┘
```

Setiap project di sini punya tujuan sederhana:

- ✅ **Solve a real problem** — walau kecil, harus ada gunanya
- ✅ **Learn something new** — selalu ada teknik/library baru yang dipelajari
- ✅ **Keep it simple** — tidak over-engineering, fokus pada inti masalah
- ✅ **Document everything** — kode yang baik adalah kode yang bisa dipahami siapapun
